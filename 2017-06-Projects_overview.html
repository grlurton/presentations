bottom-up approach to Health Information Systems analysis

A bottom-up approach to HIS gives preeminence to the local level, at every stage of the informmation chain. It is aimed at helping local producers and users of data to make best use of the limited resources and data they have.
It is different from standard top-down approaches as it does not rely on normativity of data production, nor on standardization of data analysis methods.
I explore this approach at different levels of the HIS processes :
  • Data Definition : How can global metrics used for HIV programs monitoring be adapted to local contexts and data collection practices ?
  • Data Management : Can we find a good hybridization method to maximize the value of heterogeneous data sources on local populations ?
  • Data Analysis : Can we define data analysis methods that allow programs to make an arbitrage between data quality and data collection costs ?

Aim 3 : Program Monitoring

Different approaches to data monitoring

Weighted random sampling

  • Perspective : Ex ante
  • Data Type : Panel Data
  • Data Processing : $ w_{i,t} = f(X_{I,T})$ for $T < t$ , with $w < 1 $
  • Alert rule : $A_{i,t} \sim \textrm{B}(1,w_{i,t})$
  • Reaction rule : Supervision for all sampled facilities, every facility has to be sampled every quarter.

Different approaches to data monitoring

Time series / syndromic surveillance

  • Perspective : Ex post
  • Data Type : Longitudinal Data
  • Data Processing : $ \tilde{X}_{i,t} = f(X^{v}_{i,T})$ for $T < t$
  • Alert rule : $A_{i,t} = \textbf{1}(|\tilde{X}_{i,t} - X^{o}_{i,t}| > \eta) $
  • Reaction rule : Reactive supervision, with some rule for regular supervision.

Different approaches to data monitoring

Fixed effect / league table

  • Perspective : Ex post
  • Data Type : Transversal data
  • Data Processing : $X_{i,t} = U_{i} + f(Z_{I,t})$
  • Alert rule : $A_{i,t} = \textbf{1}(\textrm{P}(U_i | \textbf{U}) < \alpha)$
  • Reaction rule : Reactive supervision, with some rule for regular supervision.

Different approaches to data monitoring

Profile Monitoring / Consistency check

  • Perspective : Ex post
  • Data Type : Mixed
  • Data Processing : $\tilde{X}_{j,i,t} = f(X_{\bar{j},I,t})$
  • Alert rule : $A_{i,t} = \textbf{1}(\sum_j(|\tilde{X}_{j , i,t} - X^{o}_{j , i,t}|) > \eta) $
  • Reaction rule : Reactive supervision
